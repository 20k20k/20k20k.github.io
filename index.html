    An anarchy server on 1.21.3 with a 20,000 x 20,000 map size. No resets, no map expansion, ever.

    How to Connect

    1. Open Minecraft Java Edition

    Launch Minecraft Java Edition version 1.9+

    2. Go to Multiplayer

    Click 'Multiplayer' from the main menu

    3. Add Server

    Click 'Add Server' and enter these details:

    Server Name: 20k20k
    Server Address: 20k20k.org

    4. Join Server

    Click 'Done', select the server, and click 'Join Server'
